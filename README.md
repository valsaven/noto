```bash
### Make a new entry:
noto "Hello, noto!"
noto 123

### Show all entries:
noto
@a9b1ah Hello, noto!
@e7sa3b 123

### Show the entry:
noto @a9b1ah
Hello, noto!

### Remove an entry:
noto -r @a9b1ah
```
